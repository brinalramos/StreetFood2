Project College - ADS 
Street Food - APP/APK
Dev: Adroind Studio - Java
A instalação do apk deve ser em um celular Android.
1º tela: Apresentação - Clique no botão "começar" para continuar. 
2º tela: Login - FUNCIONAL: Para acessar é necessario informar email e senha.
  - Utilize o link de troca de senha em caso de esqucimento.
  - Utilize o botão "Criar Conta" caso não tenha um acesso.
3º tela: Vínculo - Selecione qual tipo de acesso é o seu para realizar o cadastro.
   - Caso tenha acesso utilize o botão "voltar" para fazer o login.
4º tela: Cadastros - Preencha as iformações necessarias para prosseguir.
5º tela: Validação - Acesse sua caixa de span para validar o seu acesso.
   - Depois da validação é necessario fazer o login.
6ª tela: Tela inicial - Nesssa  tela é aprentado o mapa, com os point cadastrados no nosso app.
     - Clicando em um point sera apresentado o nome é tipo de comida que o são feitos nele.
     - Clique em Explorar para mais informações.
7º tela: Explorar -  Nessa temos os cards dos estabelecimentos cadastrados.
     - Apresenta: Nome, Status(aberto/fechado), avaliação(nota e total), km, tipo(barraca ou food truck
